# Ecolab-Mart
E-Cocommerce Web App for Eco Green Product built with Golang and Bootstrap

#Features
- Home Page
- Register, Login, Profile User
- Products Page
- Single Product Page
- Cart
- Checkout
    - Shipping Cost Calculation
    - Payment Gateway
- Order History
- Admin Panel :
    - Dashboard
    - Manage Product
    - Manage Orders
    - Manage Users
    - Reports
    - Configuration

#Tecknology Stack
- Golang:
    - Gorilla Mux
    - Gorm
- PostgreSQL
- Bootstrap
